🛠️ Build Guide
How to Build This — Step by Step
Complete beginner? No problem. Follow these steps to go from zero to live website.
Tech Stack (All Free)
HTML/CSS/JS
GitHub Pages (Hosting)
Firebase (Database)
Netlify (Optional Alt)
Git (Version Control)
Google AdSense
01
Create a GitHub Account & New Repository
Go to github.com and sign up for free. Click the + icon → "New Repository". Name it gamevault (or your brand name). Set it to Public and check "Add README". Click "Create Repository".
💡 Tip: Your free website URL will be yourusername.github.io/gamevault — make your username your brand!
02
Enable GitHub Pages (Free Hosting)
Inside your repo, go to Settings → Pages. Under "Source", select main branch and click Save. GitHub will give you a live URL in 1–2 minutes. This is your free web hosting — no credit card ever needed.
03
Download This Template & Upload It
Download this page as index.html. In your GitHub repo, click Add file → Upload files and drag in your index.html. Commit the changes. Your site is now LIVE at your GitHub Pages URL!
💡 Tip: Use VS Code (free) to edit the file locally — just open the folder and edit, then drag & re-upload to GitHub.
04
Set Up Firebase for Your Database (Free)
Go to firebase.google.com → Create Project (free). Enable "Firestore Database" in test mode. This lets users submit scores. Copy your Firebase config snippet and paste it into your index.html in a <script> tag. Firebase's free tier supports up to 50,000 reads/day — more than enough to start.
05
Get a Free Custom Domain (Optional)
Get a free domain from Freenom.com (e.g., gamevault.tk) or buy a cheap .com at Namecheap (~$10/yr). In GitHub Pages settings, add your custom domain. Change your DNS nameservers to point to GitHub. Your site goes from username.github.io to gamevault.com.
06
Add Google AdSense to Start Earning
Apply at adsense.google.com (need 10+ pages of content first). Once approved, paste the ad code snippet into your HTML. Place ads in the sidebar, between leaderboard rows, and after challenge cards. Gaming sites average $2–$5 per 1,000 views.
💡 Tip: While waiting for AdSense approval, add affiliate links from Fanatical.com or G2A immediately — no approval needed!
07
Market on Reddit, Discord & TikTok
Post in r/gaming, r/Minecraft, r/Fortnite etc. Create a Discord server and invite your first 50 users. Record a TikTok showing someone beating a challenge on your site — gaming content goes viral fast. Target 1,000 users in 30 days through consistent posting. That's enough to unlock sponsor deals.
08
Launch Pro Memberships via Ko-fi
Create a free Ko-fi.com account and set up a "$3/month" tier called "GameVault Pro". Add a "Get Pro" button to your site. Pro members get: verified badge, priority challenge slots, custom profile color. Ko-fi takes 0% — you keep everything. At 100 members = $300/month in passive income.
